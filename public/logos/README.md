# Folder Logo

Taruh file logo PNG transparan di sini dengan nama **persis** berikut.
Selama file belum ada, aplikasi menampilkan kotak placeholder (tidak error).

## Logo maskapai (otomatis dari kode penerbangan)

| File | Maskapai | Prefix kode |
|---|---|---|
| `wings-air.png` | Wings Air | WON |
| `super-air-jet.png` | Super Air Jet | SJV |
| `citilink.png` | Citilink | CTV |
| `garuda.png` | Garuda Indonesia | GIA |
| `batik-air.png` | Batik Air | BTK |
| `smart-aviation.png` | Smart Aviation | PK SNH |

> Tambah maskapai baru di `src/lib/airlineLogos.ts`.

## Logo statis desain

| File | Keterangan |
|---|---|
| `globe.png` | Logo bola dunia (pojok kiri atas) |
| `apt-pranoto.png` | Logo APT Pranoto Airport (pojok kanan atas) |

Ukuran disarankan: logo maskapai & globe ~256×256 px, logo airport ~600×208 px
(rasio lebar). Semua PNG transparan.
